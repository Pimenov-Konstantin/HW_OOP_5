# Калькулятор для вычисления арифметических выражений из двух действительных или двух комплексных чисел 

## Структура приложения:

1. класс CalcComplexExpression - вычисление арифметического выражения их 2-ух комплексных чисел
    - solveComplex - метод сложения действительных и мнимых частей арифм выражения комплексных чисел   
  

2. класс CalcRealExpression - вычисление арифметического выражения из 2-ух действительных чисел   
    - solveReal - метод для вычисления действительных чисел (части)  

3. класс ComplexNumber - формирование мнимой и действительной части комплексного числа   


4. интерфейс MathArithmeticsReal  - методы формирования арифметических операций для действительных чисел   


5. интерфейс - методы формирования арифметических операций по мнимым числам (части)   


6. класс Parser - конвертация выражения из строкового в числовой  


7. класс Presenter - логика представления данных приложения  


8. класс UI - пользовательский интерфейс с консолью  


9. класс View - ввод/вывод, проверка ввода данных приложения
    - getArithmeticReal - метод ввода арифметического выражения из 2 -ух действительных чисел
    - getArithmeticComplex() - метод ввода выражения из 2-ух комплексных  чисел 
    - checkRealExpression - метод проверки корректности ввода арифметического приложения из действительных чисел
    - checkComplexExpression - метод проверки корректности ввода арифметического приложения из комплексных чисел
